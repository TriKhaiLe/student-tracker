# Student Tracker

Ứng dụng Desktop quản lý học sinh và tự động xếp loại học lực

## Tính năng:
* **Quản lý (CRUD):** Thêm, sửa, xoá và hiển thị danh sách học sinh.
* **Tìm kiếm:** Tìm kiếm học sinh theo tên.
* **Giao diện đồ hoạ (GUI):** Thao tác trực quan qua giao diện cửa sổ Tkinter thay vì dòng lệnh.
* **Lưu trữ:** Tích hợp cơ sở dữ liệu SQLite; dữ liệu không bị mất khi đóng ứng dụng.
* **Kiến trúc OOP:** Code được tổ chức theo Hướng đối tượng, phân tách rõ ràng giữa giao diện (UI) và xử lý dữ liệu.

## Yêu cầu hệ thống (Prerequisites)
* Python 3.8 trở lên.

## Hướng dẫn Khởi chạy (How to Run)

**Bước 1:** Mở Terminal (hoặc Command Prompt / PowerShell).

**Bước 2:** Di chuyển vào thư mục chứa dự án:
```bash
cd duong_dan_den_thu_muc_project/student_tracker
```

**Bước 3:** Chạy file Python chính để mở ứng dụng:
```bash
python main.py
```

## Hướng dẫn sử dụng (How to Use)
Ứng dụng hỗ trợ 2 chế độ: giao diện đồ họa (GUI) và giao diện dòng lệnh (CLI):
 - Sau khi khởi chạy, giao diện đồ họa sẽ xuất hiện đầu tiên.
 - Sau khi tắt giao diện đồ họa, phiên bản giao diện dòng lệnh sẽ xuất hiện.
